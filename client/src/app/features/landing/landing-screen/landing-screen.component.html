Behind the scenes

Data is at the forefront of this project

In the backend, I used the most secure methods at my disposal to ensure data integrity and security by utilising best in class technologies

With an effective CI/CD pipeline through Github Actions, features can be implemented and deployed quickly. Using Microsoft Identity, user information security is taken care of with a proven framework. Fast and efficient data partitioning using Azure Cosmos for NoSQL. No relationships to query, the data is retrieved in a single read operation. Secured by the browser, cookie authentication is arguably the safest method of storing user sessions.
